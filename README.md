# 窗景 · Scenes Through Glass

一个用 WebGL 实现的极致氛围「玻璃窗风景」系统，专为 **vibe coding** 打造。

> **GitHub**: [https://github.com/Crisforlove/vibe-window](https://github.com/Crisforlove/vibe-window)

现在支持**多个不同世界**的窗外风景 + 可叠加的天气调性 + 多语言界面（中文 / English / Tiếng Việt / 한국어）。切换瞬间就能感受到完全不同的情绪。

## 快速开始

```bash
# 已经为你启动了本地服务器
open http://localhost:8787/
```

或者手动：
```bash
python3 -m http.server 8787
# 或
npx serve -p 8787
```

打开后你会看到一个全屏的雨夜玻璃窗。

## 如何进行 Vibe Coding（最爽的部分）

1. **用按钮快速切换情绪**
   - 温柔梅雨 / 深夜暴雨 / 空屋细雨 / 赛博夜雨

2. **控制台实时调参（核心乐趣）**
   打开浏览器控制台（F12），输入：

   ```js
   fx.options.mistColor = [0.012, 0.01, 0.018, 0.65]
   fx.options.refractScale = 0.72
   fx.options.dropletsPerSeconds = 180
   fx.options.gravity = 1450
   ```

   改完立刻生效！这就是 vibe coding 的即时反馈。

3. **想换背景图**
   ```js
   fx.setBackground('你自己的图片路径或线上URL')
   ```

## 推荐的第一次改动方向（挑一个开始）

- **东方诗意路线**：把折射做得像老宣纸上的水痕、加极淡的窗棂投影
- **极简治愈路线**：大幅降低雨量 + 雾气浓度，做成“窗外只有一点点雨”的安静感
- **赛博情绪路线**：让光源带彩色霓虹，雨滴折射出城市灯牌的颜色
- **带声音**：接入 Web Audio，雨滴大小影响雨声的密度和力度
- **时间流逝**：让参数随真实时间缓慢变化（从黄昏下雨 → 深夜 → 黎明）

## 项目结构

```

├── index.html          ← 你现在主要改动的地方（UI + 情绪预设 + 参数）

│   ├── bundle/index.js ← 预编译好的引擎（我们直接引用）
```
